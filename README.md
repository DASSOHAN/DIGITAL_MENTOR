Digital Mentor System A comprehensive, console-based academic tracking and mentorship management system written in C. This project leverages fundamental data structures to streamline student performance analytics, track attendance, log real-time performance alerts, and manage a peer-to-peer or teacher-assisted help queue.  🚀 Features 👨‍🏫 Teacher Module (Password Protected) Student Profile Management: Add, view, update, and delete student records seamlessly.  Dynamic Dashboard: Get an instant overview of total students, low attendance flags, class averages, and subject-wise failure counts, alongside identifying the class topper.  Performance Tracking: Automated validation of inputs (marks clamped between 0–100, attendance bound within total classes).  🎓 Student Module Personalized Reports: High-quality, formatted report cards featuring subject-wise marks, pass/fail indicators, overall grades, and automatically calculated strengths and weaknesses.  Attendance Tracker: Real-time percentage tracking with immediate visual warnings if attendance slips below the mandatory 75% threshold.  🧠 Core Data Structures Applied ⚠️ System Alerts (Stack - LIFO): Uses a Stack to store low-attendance and failing-grade warnings chronologically. The last generated alert is displayed first.  🙋‍♂️ Help Requests (Queue - FIFO): Implements a Help Desk Ticket Queue. Students can push help requests detailing a specific topic, which are served in a first-come, first-served manner by the mentor.  💾 Persistent Storage: Features automatic binary File I/O (mentor.dat) handling to ensure student records are saved on exit and cleanly loaded upon program launch.
